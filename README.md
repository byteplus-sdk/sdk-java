## sdk for byteplus

#### Install the client library
```xml
<!-- https://mvnrepository.com/artifact/com.volcengine/byteplus-sdk -->
<dependency>
    <groupId>com.volcengine</groupId>
    <artifactId>byteplus-sdk</artifactId>
    <version>${latest}</version>
</dependency>
```
